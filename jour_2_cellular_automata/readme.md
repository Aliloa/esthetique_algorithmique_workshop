# Présentation
- Projet réalisé en JavaScript avec la bibliothèque p5.js

Le projet consiste en un petit jeu 2D pixelisé appelé The Floor Is Lava.
Une map est générée procéduralement à l’aide d’un automate cellulaire.
Le joueur peut se déplacer sur les cases libres pendant que le feu (également généré sur le principe d'automate cellulaire) se propage progressivement sur la map.

Le but est de survivre le plus longtemps possible sans toucher le feu.

## Installation et exécution

- Ouvrir le fichier index.html
