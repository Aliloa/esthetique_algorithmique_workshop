# Présentation
Le projet a été fait en html css javascript

C'est une petite histoire et avec les choix à la fin ça determine le type du joueur en se basant sur la grille d'alignement (de donjons et dragons)

J'ai ajouté une gestion de modèles 3D. Des choix qui dépendent des pièces.
Et avancé l'histoire
